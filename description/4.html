Seguros de Vida

Descripción

  • 14-01-2018

Es el módulo central de Kinetyc PAS Life. Se hace cargo de todo el proceso de manejo de cotización, propuesta, suscripción, emisión, plan de pago y endoso. AGL contiene la base de pólizas y clientes asociados a ellos.

El proceso de emisión se reduce a una secuencia de cuatro pasos: Cotizar → Ingresar datos de propuesta → Ingresar datos de pago → Suscribir y emitir.
