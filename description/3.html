Este mini proyecto nace de uno de las tantas tareas dadas en el curso de Angular impartido por Escalab. De todos los tuve que desarrollar es mi favorito ya que, había que desarrollar un proyecto de libre pensamiento, utilizando Angular y consumiendo la API de “The Rick and Morty”. ¿Por qué me gustó? Las razones son simples. Una de ellas es que, por fin después de mucho tiempo, pude “jugar” con el diseño de un sitio web usando HTML y CSS. Busque y agregue distintos package para mejorar un poco el sitio, entre ellos, un paginador para los personajes de la serie y un swipe, que lo utilice como banner dentro de la página web. Por último, investigue y aplique Firebase hosting para dejar una demo del sitio, así cualquiera podría verlo sin la necesidad de descargar el código fuente desde mi GitHub.
