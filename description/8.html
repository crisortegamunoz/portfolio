Campañas Comerciales

Descripción

  • 14-08-2020

Parte del core de Kinetyc. Permite la creación de campañas comerciales en base a un convenio comercial previamente configurado. Esto permite a la misma compañia de seguros disponibilizar una serie de productos "en oferta" para ofrecerla a sus clientes, así como también permite a un intermediario entregarle un enlance donde los clientes puedan ver y cotizar ciertos productos.
