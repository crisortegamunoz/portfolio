Seguros Líneas Comerciales

Descripción

  • 15-12-2016

Es un módulo dentro de Kinetyc Platform orientado a los seguros de garantía, es utilizado por los intermediarios internos de una compañia. Se hace cargo de todo el proceso de manejo de cotización, propuesta, suscripción, emisión, plan de pago y endoso. AGL contiene la base de pólizas y clientes asociados a ellos.
