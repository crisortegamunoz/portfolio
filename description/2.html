Esta fue una versión de mi portafolio en que invertí mucho tiempo. Apliqué todo el conocimiento adquirido en los cursos de Angular (Platzi y Escalab). Utilizando Angular Material, Firebase Hosting para tener el sitio publicado en la web, Firebase Storage para guardar las imágenes y Firebase Database para guardar la información que se mostraría en el sitio. Es un poco triste el hecho de perder mi cuenta principal de Github donde tenía el código para realizar modificaciones o upgrades. Tengo pensado volver a hacerlo, pero utilizando otras tecnologías.
